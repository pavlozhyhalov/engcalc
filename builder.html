Конфігуратор нестандартного конвеєра

Зберіть схему свого механізму крок за кроком — від типу привода до несучого елементу. Калькулятор побудує модель і розрахує всі вузли.

01
Тип привода
Джерело руху
Мотор-редукторЄдиний блок. Компактний, простий монтаж. Найпоширеніший варіант.
Двигун + окремий редукторБільший момент, ремонтопридатність, гнучкість компоновки.
ГідромоторПлавне регулювання, великий момент на низьких обертах.
Двигун + частотний перетворювачРегульована швидкість, плавний пуск, захист від перевантаження.
02
Конфігурація приводного вала
Передача руху на тяговий орган
Поперечний вал + 2 зірочкиВал поперек конвеєра з зірочками по краях. Для двох паралельних ланцюгів.
Одна центральна зірочкаОдин ланцюг по центру. Просто, але можливий перекіс несучої частини.
Приводний барабанДля стрічки або суцільної сітки без ланцюгів. Передача тертям.
Два синхронізованих приводиПо приводу на кожен ланцюг. Широкі та важкі конвеєри.
03
Тяговий орган
Елемент, що передає зусилля
Роликовий ланцюг (ISO 606)Серії 08B–32B. Компактний, точний крок, доступний.
Конвеєрний ланцюг (DIN 8167)Подовжений крок, кріплення для поперечних елементів.
Стрічка (гума / ПВХ / ПУ)Для барабанного привода. Тихий хід, харчове виконання.
Сталевий канатПідвісні та канатні системи, великі прогони.
04
Несучий елемент
Що безпосередньо несе вантаж
Поперечні прутиПрути з'єднують два ланцюги. Відкрита конструкція, дренаж.
Металева сіткаПлетена або зварна на прутах. Харчова, термообробка, мийка.
ПластиниСуцільний настил. Важкі та гострі вантажі.
ПланкиПоперечні планки з зазором. Вентиляція.
КовшіПідйом сипучих. Елеваторна схема.
СкребкиШтовхають матеріал по жолобу.
05
Напрямні / опори тягового органу
Визначає коефіцієнт тертя
Ковзання по сталіf ≈ 0.15. Просто і дешево, знос при абразиві.
Нержавійка по нержавійціf ≈ 0.20. Харчові виробництва без мастила.
UHMW-PE напрямніf ≈ 0.08. Тихий хід, хімічна стійкість.
Роликовий ланцюг по рейціf ≈ 0.05. Мінімальне тертя, висока точність.
