# ConveyorCalc

Інженерний калькулятор транспортних механізмів — конвеєри, елеватори, конфігуратор нестандартних схем. Розрахунок за ISO 5048, DIN 22101, CEMA 350, FEM 2.122, ISO 606, ДСТУ.

## Структура

```
index.html      — головна: каталог 18 типів механізмів
calc.html       — сторінка розрахунку (тип через ?type=, 3 вкладки: розрахунок / джерела / приклад)
builder.html    — конфігуратор нестандартного конвеєра
css/style.css   — дизайн-система
js/calc.js      — методики розрахунку, джерела, перевірені приклади
js/builder.js   — логіка конфігуратора
```

## Реалізовані методики

| Тип | Стандарт | Приклад перевірено за |
|---|---|---|
| Стрічковий | ISO 5048 / DIN 22101 | Спиваковський, 1983 |
| Шнековий | CEMA 350 | CEMA 350 4th Ed. |
| Ланцюговий із сіткою/прутами | ISO 606 / DIN 8167 | Tsubaki Guide to Chain |
| Скребковий | CEMA 550 | Спиваковський, гл. 7 |
| Роликовий | Interroll Manual | Interroll Manual |
| Ківшевий елеватор (стрічка) | FEM 2.122 | Паспорт норії НЦ-50 |

Решта типів — заглушки, методики додаються у `js/calc.js` (об'єкт `CALCS`).

## Деплой на GitHub Pages

1. Створи репозиторій, залий файли в корінь
2. Settings → Pages → Source: Deploy from a branch → main → / (root)
3. Сайт буде на `https://<username>.github.io/<repo>/`

Бекенд не потрібен — сайт повністю статичний.

## Додавання нового типу розрахунку

У `js/calc.js` додай ключ до об'єкта `CALCS`:

```js
my_type: {
  title: 'Назва',
  subtitle: 'Опис',
  form: `<html форми>`,
  run() { /* розрахунок, виклик showResults(html) */ },
  sources: `<html джерел>`,
  example: `<html прикладу>`
}
```

І плитку в `index.html` з посиланням `calc.html?type=my_type`.
